OUR WEBSITE PROJECT

Plan Overview

Our team started working on building this website as soon as the assignment was published by the instructor. We created a public repository in github and all of us joined it. We divided the website into 4 sections: Main page and Team Members, Hour of Code, Code Art, Web Project. In order to help each other and ensure that everything is on track, we met 3 times a week. Rest of the days we constantly kept each other updated on the steps we took. Our team designed this website using HTML for structure, CSS for styling, and JavaScript for dynamism. For each HTML file, we added seperate CSS files and used different extensions in VS Code to make the process easier for us. The images on the left are screenshots of the code we created and the texts next to them are explanations for how it works.

Homepage

The code for this section represents a structured home page for our website titled "Phoenix Team”. The head tag includes the webpage's title, "Homework 3: How websites work?", and links to an external CSS file (index.css) for styling and a JavaScript file for interactivity. In the body section, a navigation bar provides links to different pages, such as "Home," "Our Team," and "Web Project," using an unordered list. Each list item contains an anchor tag for navigation, directing users to specific HTML or CSS files. The main content displays a logo image, the site name inside an h2 tag, and a description within a paragraph tag. This description uses bold text (strong). Lastly, the footer section credits Ada University.

Our Team

In this HTML code, each team member is presented within a division tag using the class team-member, containing our photos, name, description, and contact links represented by icons. These icons use anchor tags for clickable links, and the href attribute specifies their destination, such as email, phone, or LinkedIn profiles. Additionally, a button element with the ID scrollToTopBtn provides a "Scroll to Top" functionality, controlled via JavaScript. External styles (team-members.css) and JavaScript are linked through link and script tags, enabling the integration of custom styles and interactivity.

Hour of Code

This HTML code explains our past Hour of Code event. It includes sections for a carousel, event details, and a summary of the event. The body contains a primary section with id="hour-of-code" for the main content.The carousel is implemented using radio buttons (input type="radio") for navigation between six slides, each containing an image tag to display event photos. Corresponding label tags create navigation dots. A container organizes paragraphs detailing the event. The details section uses an unordered list for quick facts like location, date, and participants. Additional sections (h3 and p) provide a narrative on the event's introduction, progress, and conclusion.
